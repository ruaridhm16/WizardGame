# Unnamed Wizard Game

A deck-building game in progress. Cards combine in different ways to create different effects - build your deck, chain combinations, cast spells.

Status: work in progress.

## About this repo

This repository is for team collaboration during development - it is **not set up to be run or played**. Feel free to browse the code.

Built with Unity, all scripts in C#.
